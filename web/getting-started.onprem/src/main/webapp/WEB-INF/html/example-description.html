This example demonstrates the use of the Pipeline API to perform IP Intelligence lookups in a simple Web Application servlet project.

In particular, it highlights

  1. Automatic processing of IP addresses from HTTP requests;
  2. Custom IP address lookup via form input;
  3. Display of location, network registration, and geographic data.

The example GettingStartedWebOnPrem uses on-premise IP Intelligence data files for fast, local processing without external API calls.

This example demonstrates use of the PipelineFilter, configured via the web.xml and pipeline options 51Degrees-OnPrem.xml files, to create a pipeline using on-premise IP Intelligence.
