Combined device detection and IP intelligence example

This example demonstrates the use of the Pipeline API to perform both device detection and IP intelligence lookups within a single Java web project. It highlights device detection using User-Agent and Client Hints, IP-based geolocation and network information, client-side evidence collection for enhanced detection, and parallel processing of both engines for optimal performance.
